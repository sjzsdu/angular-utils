Event Trader

股票交易策略系统

本项目旨在提供一个灵活的股票交易策略框架，允许用户通过多种策略对股票进行分析和交易

项目特点

安装与使用

pip install event_trader
 
from event_trader.strategies import MA2Strategy, KDJStrategy
from china_stock_data import StockData

stock = StockData('601688')
for strategy in [MA2Strategy, KDJStrategy]:
    st = strategy(stock)
    st.show(transaction=False, days=30)
